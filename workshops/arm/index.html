DESCRIPTION :

Most new cars and 95% mobile phones contain microcontroller chips designed by ARM, leading semiconductor design company. In fact, ARM architecture is used in embedded systems of almost all modern electronic devices. This workshop is designed to provide industry-level experience in developing embedded systems projects using the widely used ARM architecture.

Topics Covered

• ARM architecture and programming
• mbed API
• I2C and USB communications
• Self-balancing robot- Overview and concepts
• Programming – Embedded C++

Eligibility
